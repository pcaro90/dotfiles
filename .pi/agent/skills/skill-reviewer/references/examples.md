# Skill Review Examples

Read this file when a reviewed skill defines output formats, transforms input to output, or needs concrete examples.

## Exact output template pattern

```markdown
## Report structure
ALWAYS use this exact template:
# [Title]
## Executive summary
## Key findings
```

Flag output format descriptions that say "something like" or "similar to" instead of providing an exact template.

## Input/Output example pattern

```markdown
**Example 1:**
Input: Added user authentication with JWT tokens
Output: feat(auth): implement JWT-based authentication
```

Transform-style skills such as formatting, conversion, data extraction, and code generation usually benefit from at least one representative Input/Output example.
